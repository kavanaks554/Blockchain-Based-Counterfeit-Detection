# Blockchain-Based Counterfeit Detection System

## Overview
This project is a blockchain-based system designed to detect counterfeit cosmetic products using QR code validation and secure database verification.

## Features
- QR code-based product authentication
- Manufacturer, Seller, and Customer modules
- Secure database integration
- Blockchain-based record verification

## Technologies Used
- Python
- SQL (HeidiSQL / MySQL)
- Blockchain concept

## How It Works
1. Manufacturer registers a product.
2. A unique QR code is generated.
3. Product data is stored in the database and blockchain.
4. Customer scans the QR code to verify authenticity.
